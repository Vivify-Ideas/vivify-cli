## Description

[VivifyScrum](https://www.vivifyscrum.com/) CLI tool

Available Actions:
- Show Board (Persisted locally),
- Show Loaded Board Item (from locally persisted Board),
- Create Board Item,
- Update Board Item,
- Remove Board Item

More actions will be added soon...
